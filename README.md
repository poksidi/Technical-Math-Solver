# Technical Math Solver

Aplikasi edutainment matematika teknik berbasis web untuk siswa SMK, dibangun menggunakan Flask.

## Fitur Utama

1. **Konversi Satuan** - Konversi berbagai satuan panjang, massa, dan suhu
2. **Perhitungan Kelistrikan** - Hukum Ohm, rangkaian seri-paralel
3. **Perhitungan Material** - Volume, berat, analisis stress sederhana
4. **Geometri Teknik** - Perhitungan bentuk 2D dan 3D

## Teknologi yang Digunakan

- **Backend**: Flask (Python)
- **Frontend**: HTML, CSS, JavaScript, Bootstrap 5
- **Storage**: JSON file untuk riwayat perhitungan
- **Icons**: Font Awesome
